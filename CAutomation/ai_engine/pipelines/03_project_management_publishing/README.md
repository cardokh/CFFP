# Pipeline 03 – Project Management Publishing

This pipeline publishes approved planning artifacts from Pipeline 02 to configured project management tools.

## Responsibility

Pipeline 03 owns publication only. It consumes the frozen Planning Package and publishes its approved agile planning artifacts to external project management systems according to configured adapters.

## Boundaries

- create or change planning content
- generate source code
- validate generated artifacts
- apply files to the repository
