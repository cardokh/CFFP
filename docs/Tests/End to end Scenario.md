 Simple end-to-end scenario

1. Log in as Admin.
2. Create a Lesson Category.
3. Create two Learning Items.
4. Create two Quiz Questions.
5. Create a Lesson and attach the Learning Items and Quiz Questions.
6. Create a User.
7. Assign the Lesson to the User.
8. Log in as that User in both Desktop/Mobile Interfaces
9. Verify that the assigned Lesson appears in Lesson Library.
10. Open Lesson Details and verify the Learning Items.
11. Start the Lesson.
12. Complete the Lesson and Quiz.
13. Verify that the result appears in Progress.

If this flow works successfully, it demonstrates that the main application workflow 
functions end-to-end from administration through learning and progress tracking.
