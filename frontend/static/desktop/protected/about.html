HejSan

HejSan is a Swedish learning companion built for short, focused language practice across desktop and mobile experiences.

Project

This application was developed as part of the User Interface Programming and Evaluation course at Uppsala University.

The platform supports lessons, learning items, quizzes, progress tracking, and robot-inspired interaction styles.
