# Applications

## Purpose

This folder contains applications that use the Epic Tracker architecture.

The first planned application is the Pipeline Management System. It is documented as the first reference application for the factory, but it is not implemented in this iteration.

## Responsibilities

- Group application-specific epics, user stories, implementation tasks, test suites, test cases, validation output, and reports.
- Keep application concerns separate from reusable Epic Tracker module conventions.
- Provide a place for future real implementation artifacts without changing the module-level architecture.

## Current Scope

Documentation and folder structure only. No business logic or code generation is included.
