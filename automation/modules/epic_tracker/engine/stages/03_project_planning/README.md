# Project Planning

This is a generic reusable pipeline stage.

## Purpose

Produce application-specific Agile planning artifacts such as epics, user stories, tasks, test suites, and test cases.

## Application data

Application-specific input, output, validation results, and reports are stored under:

```text
applications/<application_name>/stages/03_project_planning/
```
